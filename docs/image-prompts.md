## Usage Notes

1. Generate images at highest resolution available
2. Save as `.jpg` (photos) or `.png` (transparent backgrounds)
3. Optimize with TinyPNG before adding to repo
4. Place all files in `frontend/public/images/`
5. Reference in code as `/images/filename.jpg`
